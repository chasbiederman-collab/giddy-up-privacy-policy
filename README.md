# giddy-up-privacy-policy

Privacy Policy for GiddyUp Farrier, ready for GitHub Pages.

## Files

- `index.html` - public privacy policy page

## Publish with GitHub Pages

1. Go to **Settings** → **Pages**
2. Under **Build and deployment**:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
3. Save

Your policy URL will be:

`https://chasbiederman-collab.github.io/giddy-up-privacy-policy/`
